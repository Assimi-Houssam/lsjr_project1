1 Accéder à la caméra avec getUserMedia
La méthode navigator.mediaDevices.getUserMedia permet de demander l’autorisation d’accéder à la caméra du navigateur.

2 Afficher le flux vidéo de la caméra
On utilise le flux (media stream) obtenu pour afficher la vidéo en direct dans une balise <video>.

3 Utiliser une bibliothèque de lecture de QR code (qr-scanner)
https://github.com/nimiq/qr-scanner
 simplifie l’analyse du flux vidéo pour détecter des QR codes.

4 Décoder le QR code
Lorsque la bibliothèque détecte un QR code dans le flux vidéo, elle renvoie automatiquement le texte décodé (par ex. un identifiant, un lien, etc.).

5 Envoyer l’ID au backend
qr code -> have id -> send id as param to backend -> return data
